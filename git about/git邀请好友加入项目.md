# git邀请好友加入项目

1、进入项目点击Settings。

2、点击**Collaborators**，Add people。输入被邀请人账号。

3、再点击**Select a collaborator above**。

![image-20230424103520609](D:\fantai\JAVA\notes\git about\git邀请好友加入项目\image-20230424103520609.png)

4、被邀请人确认加入项目。
